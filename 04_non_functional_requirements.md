# NFR-1. Производительность  
Бот должен обрабатывать запрос пользователя и отправлять домашнее задание не более чем за 2 секунды с момента получения команды при нормальной нагрузке.

# NFR-2. Доступность  
Сервис должен быть доступен пользователям 24/7, за исключением времени проведения технического обслуживания.

# NFR-3. Масштабируемость  
Система должна поддерживать одновременную работу не менее 1000 пользователей без существенного снижения скорости ответа и потери сообщений.

# NFR-4. Безопасность  
Доступ к функциям просмотра и получения домашних заданий должен предоставляться только авторизованным пользователям Telegram по их Telegram ID.

# NFR-5. Юзабилити  
Интерфейс бота должен быть простым и интуитивно понятным: пользователь должен иметь возможность получить домашнее задание с помощью коротких команд и понятных кнопок без дополнительного обучения.
