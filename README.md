# 🎓 NAKOL

NAKOL (Notification And Knowledge Organizer for Learning) — Telegram-бот, который превращает скучное отслеживание домашки и расписания в игру с мемами. Никаких скучных списков — только дедлайны, напоминания и награда за выполнение.

## 🚀 Elevator Pitch
NAKOL — это умный помощник для студентов, который автоматически собирает расписание и домашние задания из фото, Excel или ручного ввода, присылает напоминания о дедлайнах и выдаёт мемы за каждую выполненную задачу. Всё, чтобы учёба была организованной и чуточку веселее.

---

## ❗ Проблема и решение

**Проблема:**  
Студенты постоянно забывают о дедлайнах, теряют домашку в чатах, а старосты тратят часы на оповещение группы. Мотивации делать вовремя нет, а напоминания в общих чатах просто теряются.

**Решение:**  
NAKOL даёт старостам удобные инструменты импорта (фото, Excel, ручной ввод), автоматически рассылает задания всем участникам группы, присылает персонализированные напоминания и мотивирует мемами за каждое выполненное задание.

---

## 👥 Целевая аудитория

- **Старосты** — которым надоело собирать домашку и писать в чаты.
- **Участники групп** — которые хотят получать все задания в одном месте и не пропускать дедлайны.
- **Индивидуальные пользователи** — кто учится один и хочет личный планировщик с мемами.

---

## 🔥 Ключевые преимущества

1. **Импорт за 5 секунд** — сфотографировал доску с домашкой, бот сам распознаёт текст и отправляет группе.
2. **Мемы за выполнение** — закрыл задачу → получил котика. Простая, но рабочая мотивация.
3. **Умные напоминания** — за 24 часа и за 3 часа до дедлайна, с мотивирующими подписями.
4. **Два режима** — работа в группе со старостой или полностью индивидуально.

---

## ⚙️ Главные возможности (команды бота)

| Команда | Описание |
|---------|----------|
| `/start` | Выбор роли: староста, участник группы, индивидуальный пользователь |
| `/import_schedule` | Загрузить расписание (Excel / фото / вручную) |
| `/import_homework` | Загрузить домашние задания (фото / вручную) |
| `/list` | Показать мои задания с сортировкой по срочности |
| `/done [ID]` | Отметить задание выполненным и получить мем |
| `/remind` | Настроить напоминания (время, тип) |

---

## 🛠 Технологии

- **Бот:** Python + aiogram 3.x
- **База данных:** SQLite (MVP) / PostgreSQL
- **Распознавание текста:** EasyOCR (русский + английский)
- **Парсинг Excel:** openpyxl / pandas
- **Фоновые задачи:** apscheduler
- **Тестирование:** pytest
- **CI/CD:** GitHub Actions
