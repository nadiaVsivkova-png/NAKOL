# User Story
## 1. **Как** студент, **я хочу** иметь быстрый доступ к домашним заданиям, **чтобы** экономить время на их поиске.
**Критерии приемки**:
* Бот выводит список задач по запросу.
## 2. **Как** студент, **я хочу** выбирать, в какое время мне удобнее получать д/з, **чтобы** напоминание пришло вовремя и я его не потерял.
**Критерии приемки**:
* Бот по запросу пользователя дает возможность изменить настройки напоминаний.
## 3. **Как** студент, **я хочу** получать позитивные/мотивирующие фразы вместе с д/з, **чтобы** продуктивнее работать.
**Критерии приемки**:
* Бот напряавляет в сообщении в напоминаниями позитивные/мотивирующие фразы.
## 4. **Как** староста или индивидуал, **я хочу** заносить д/з 1 раз во всеобщий доступ, **чтобы** потом не тратить время на поиски и напоминания.
**Критерии приемки**:
* Староста может отправить д/з в группу.
* Группа получила задания, направленные старостой.
* Индивидуал может создать личное расписание. 

## 5. **Как** староста или индивидуал, **я хочу** составлять  расписание, **чтобы** помещать туда задачи.
**Критерии приемки**:
* Индивидуал может создать личное расписание.
* Староста может создать расписание для группы.

## 6. **Как** студент, **я хочу** отключать и включать напоминания, **чтобы** в свободные дни они меня не отвлекали.
**Критерии приемки**:
* Студент может временно отключить уведомления.
* Студент может включить уведомления.
## 7. **Как** студент, **я хочу** получать мемы за сделанные задания, **чтобы** поднять настроение.
**Критерии приемки**:
* Бот отправляет мем студенту при отмеченном выполненном задании.
## 8. **Как** студент, **я хочу** запросить д/з с ближайшим дедлайном в свободное время, **чтобы** освободить себе время на потом.
**Критерии приемки**:
* Студент в любое время может запросить д/з.
* Система выводит д/з с ближайшими дедлайнами.
## 9. **Как** староста, **я хочу** отправить сообщение всей группе, **чтобы** передать важные новости.
**Критерии приемки**:
* Староста может отправить сообщение всей группе.
* Система отправит сообщение каждому участнику группы.
## 10. **Как** староста или индивидуал, **я хочу** удалить предмет, который закончился, из расписания, **чтобы** в расписании были только актуальные данные.
**Критерии приемки**:
* Для старосты и индивидуала реализована функция удаления предмета.
* При удалении предмета данные о нем удаляются из базы данных.
## 11. **Как** староста или индивидуал, **я хочу** загрузить расписание сессии, **чтобы** оно всегда было в быстром доступе.
**Критерии приемки**: 
* Студенты могут посмотреть расписание сессии в системе.
* Староста и индивидуал могут загрузить расписание сессии в системе.
## 12. **Как** система, **я хочу** удалять задачи с дедлайном, истекшим более двух недель назад, **чтобы** не засорять базу данных и не показывать неактуальные дедлайны.
**Критерии приемки**:
* Задачи с дедлайном истекшим более месяца назад удаляются из системы.
## 13. **Как** студент, **я хочу** получить весь функционал системы по одному запросу, **чтобы** знать знать, какие опции я имею.
**Критерии приемки**:
* Студенты могут по одному запросу получить список всех доступных функций системы.

## 14. **Как** студент, **я хочу** зарегистрироваться в системе, **чтобы** облегчить свое обучение.
**Критерии приемки**:
* Пользователь может зарегистрироваться в системе. 

## 15. **Как** пользователь, **я хочу** получать напоминания о задачах, **чтобы** ничего не забывать.
**Критерии приемки**:
* Пользователь получает уведомления о дедлайнах. 

## 16. **Как** студент, **я хочу** посмотреть расписание в системе, **чтобы** знать свой распорядок дня.
**Критерии приемки**: 
* Студент может посмотреть расписание.

## 17. **Как** студент, **я хочу** посмотреть расписание сессии в системе, **чтобы** знать дни экзаменов.
**Критерии приемки**: 
* Студент может посмотреть расписание сессии.

**В User Story, в которых не указана конкретная роль, под студентом может быть любая роль (староста, индивидуал, участник группы)**

**Во всех User Story под системой подразумевается ТГ-Бот NAKOL**
